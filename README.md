# Project-02-Website-with-a-Search-Engine
# Author: Jordan Neslony

# Description:
This website aims to provide users with accurate gifs corresponding to the text answer they submitted in the search bar.

# Functionality: 
- This project features a nav-bar at the top of the screen that is navigatable
 via hyperlinks.
- The main feature of this website is its search bar and submit button. Upon typing in a phrase it will send a request to giphy via its api and then return images that will display on the screen for the user to peruse.
- Additionally this website is accesible for mobile devices and lower screen resoulution.

# Technologies: HTML, CSS, SCSS, Bootstrap, and JavaScript.

# User Stories:
1. As a parent, I want to find funny and relatable gifs, so that I can entertain my children over text.
2. As a friend in an online group chat, I want to send funny and on-topic memes, so that my friends and I can laugh.
3. As a statistician, I want to see the gifs related to a specific name, so that I can see the impact internet culture has had on gifs.
